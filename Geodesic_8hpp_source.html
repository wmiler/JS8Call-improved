62 // Distance class, describes a distance in kilometers. Created via
63 // interpolation of Maidenhead grid coordinates, and as such will
64 // be invalid if interpolation failed, typically due to bad coordinates.
65 //
66 // May additionally be defined as 'close', meaning that either of the
67 // grids provided was only 4 characters and the computed distance was
68 // short, so we know it's close, but not just how close. In this case,
69 // the value will be a non-zero minimum constant, and string conversion
70 // will prepend a '<'.
71 //
72 // While distances are stored internally only in kilometers, caller may
73 // request string conversion in terms of statute miles.
